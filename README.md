## Amphibians App (양서류 앱)
An app communicates with Server. (서버와 통신하는 앱입니다)   
Take json which have information about Amphibians from server, and shows it to users. (서버로부터 양서류 정보가 담긴 json을 받아 사용자에게 보여줍니다.)
